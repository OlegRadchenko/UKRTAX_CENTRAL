Цей репозиторій реалізовано за спільним Чесько-Українським проектом "Strengthening the forest monitoring capacity of Ukraine for EU compliance".
Проект UAFMDR містить програмні засоби, які забезпечують централізоване зберігання та використання даних лісовпорядкування, які збираються додатком Field-Map UKRTAX.
Описи реалізованих засобів розподілено по папках сегментів (вкладених до папки Segments):
  - mca: Засоби для загального управління та аудиту;
  - ds: Основні засоби зберігання та обробки даних центразізованого сховища;
  - ukrtax: специфічні засоби для додатку Field-Map UKRTAX
Описи структур зберігання даних, які використовуються сегментами, розміщено у підпапках Tables кожного сегменту.
Головна зовнішня функційність реалізується збереженими процедурами:
  Оновлення налаштувань структури сховища для роботи з додатком UKRTAX: ukrtax_update_app_structure 
  Імпорт даних із оперативної бази Field-Map UKRTAX: ukrtax_import_fm_database
  Створення оперативної бази Field-Map UKRTAX з центрального сховища: ukrtax_export_data
Службові об'єкти та функції реалізовано у класах UAFMDRHelper та UKRTAXHelper.
